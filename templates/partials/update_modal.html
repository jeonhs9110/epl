Update Mode

Two options — lightweight refresh (scrape only) or full retrain (scrape + spin up a GPU VM for training, auto-deletes itself when done).

Full Retrain (recommended for weekly run)
  1. Scrape results for all 13 leagues (only new matches)
  2. Scrape upcoming fixtures for next 30 days
  3. Push fresh CSVs to Cloud Storage
  4. Create a GPU VM (L4 Spot, asia-northeast3) automatically
  5. GPU trains DL + Hybrid + Calibration + PPO with real-time logs streaming here
  6. GPU pushes new models back to this CPU VM and self-deletes
~2.5–3 hours total. Cost ≈ $0.40 per run. You can close this modal and come back — training continues regardless.
Scrape Only — for daily data refresh
Refreshes match data without retraining. Uses the existing model for predictions. Runs automatically every day at 11:00 KST.
